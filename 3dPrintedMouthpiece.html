3D Printed Saxophone Mouthpiece Project

The purpose of this project is to use 3D modeling tools and 3D printing to design and create custom saxophone mouthpieces. The shape, particularly internally, can then be easily modified to experiment with different designs and see the results and the effect different modifications have on the sound and playability of the mouthpiece.


I have tried several times to model a saxophone mouthpiece, but have hit a barrier each time. The shape of a saxophone mouthpiece has proven particularly challenging to model, and especially to model accurately (which is important). I finally figured out a way to create the model which not only produces an accurate recreation of a mouthpiece, but also allows different parameters of the model to be varied fairly easily.


I set up the model as a set of boolean operations. There is one shape that serves as the base shape for the mouthpiece, then several others that subtract volume from that base shape. Using this setup, any of the subtracting pieces can be modified and those changes will be reflected in the base.


There is also a "subdivision surfaces" modifier on all the objects, which increases their resolution and smooths them, making it simpler to create a high resolution mesh without needing to edit as many vertices.





Below are two prototypes I 3D printed. The blue one is the original, which is based soley on the Meyer mouthpiece, as closely as I could match it. The clear one is a modification of that model, where the slope of the inner part of the mouthpiece is changed to have a much more distinct point where it goes from a shallow angle to a steep angle. This is called a "baffle", and should lead to a much "brighter" sound.


As an aside, this is a mouthpiece that I carved from a branch of poplar. I used a saw to cut it to the general shape, and then used mainly a whittling knife (although occasionally a Dremel and a drill to carve out the middle). It is much less precise than any mouthpiece I own or have 3D printed, but it does play!
