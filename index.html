3D Printed Mouthpiece Project
Last updated: September 2015

Experimenting with modeling saxophone mouthpieces (in Blender) and 3D printing them. Looking to see the effects of different internal and external shapes on sound characteristics.

Motor Testing Apparatus (Steady State 'Brake' Dynamometer)
Last updated: September 2015

Motor testing apparatus for the ENES100 course at UMD. Allows for testing to get a Torque vs. Angular Rotation curve to characterize motors (to confirm manufacturer numbers or identify unknown values).

Personal Quadcopter
Last updated: September 2015

My personal quadcopter, assembled from custom picked parts. It is a 250mm size UAV, running on a Naze32 flight control board, using Baseflight for configuration.

DCC Capstone Project
Last updated: September 2015

This was a project to convert 3D models into a 3D-printable 'building set'. Written in Ruby, it converts a 3D model into a set of connectors that can then be 3D printed and assembled to form a wireframe version of the model. This was my capstone project for the Honors College I was in, Digital Cultures and Creativity.

Quadcopter Project
Last updated: May 2014

Project that I am working on with two fellow undergraduate students. We are attempting to set up First Person View (FPV) flight as well as video recording, control it via laptop (and possibly eventually Xbox Kinect motion sensing), and modify the software to hook up sensors/peripherals.
